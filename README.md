# About
This repository contains the code for our CS 5100 Foundations of AI project at Northeastern University (Boston). We’re building an NLP-powered web app that translates between English and a lower-resource language (currently considering Korean/Hangul) using fine-tuned Hugging Face models, with support for direct text input, PDF or text-based file uploads, and images via OCR, all within an easy-to-use GUI built using either Gradio or Streamlit.

If time permits, we may also add support for audio-based input using speech-to-text models.
